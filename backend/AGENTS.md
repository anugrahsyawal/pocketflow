# Backend Instructions

- Backend implementation has not started. This directory is reserved.
- Do not select a stack, scaffold services, add dependencies, or define live
  infrastructure without Product Owner approval.
- Read the canonical product requirements, decision log, backlog, current
  sprint, and any approved API/data contracts before implementation.
- Production authentication is required before production; provider and
  session model are TBD.
- Remote synchronization, archive/delete propagation, idempotency, conflict
  handling, migrations, and backup/restore contracts must be decided before
  implementation.
- Permanent deletion is an explicit user action. Backend tombstones, retention,
  and audit behavior remain TBD.
- Deployment platform, backup retention, encryption, storage location, and
  restore-test frequency remain TBD. Do not infer a provider.
